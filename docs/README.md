# LxBox
> 基于LSE的ll空岛插件核心

## 初衷
 > 1.快速开发空岛插件，开发了LxBox插件核心，该插件核心提供了一些基础功能，如：组队、保护、经济、消息<br>2.使用该框架，能够实现子插件的快速开发，与功能的自选化，使服主能够自定义自己想要的空岛插件<br>3.该框架提供了一套简便API，将一些功能整合起来，也可以成为开发其他插件的工具
